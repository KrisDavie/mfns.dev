Here you'll find a list of tools and projects I've built or worked on, mainly for the A Link to the Past Randomizer.

Glitch Maps
A collaboration with malmo. Interactive maps displaying a wide variety of glitches possible in A Link to the Past including descriptions and links to text/image/video guides.
Door Tracker
An autotracking capable door tracker for ALttPR Door Shuffle. Builds dungeon maps and shows the player's location.
Modified Dunka Tracker
A modified version of Dunka Tracker which includes visual changes to the map [StructuralMike], green keys [Telethar], glitched logic (updated from otherTom's implementation), a customizable color scheme and non-ladder based presets. Often updated without warning!
ALttPR Patcher A standard patcher for ALttPR which also allows the renaming of players within multiworld seeds. Secret Patcher An extension of the above tracker which also has the ability to patch non race-legal settings and gameplay options, including: Enabling Pseudoboots, Disabling FastROM, Re-introducing item collection lag (removed with the decompression update), and removing fanfare from item pickups (currently causes no rupees to be collected from chests). Timeline Tracker (WIP)
A tracker that displays the items collected, and bosses defeated in a timeline format. Includes a timer which can be linked to an upcoming/ongoing ladder or rt.gg race, or can auto start/end based on the game state [not 100% accurate]. This is a work in progress.
